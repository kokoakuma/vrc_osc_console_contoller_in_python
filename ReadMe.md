## OSCで前後左右に移動できるコンソールアプリです。

### 実行環境
- Windows 10
- Python 3.9.10
- VRChat 2022.1.1

### 実行方法
1. 任意のディレクトリ（以下、該当ディレクトリ）にダウンロード
2. Pythonダウンロード
3. コマンドプロンプトから必要なパッケージをインストール
`python -m pip install python-osc`
4. 該当ディレクトリに移動後、コマンドプロンプトから実行
`python main.py`
5. h, j, k, lを入力で前後左右移動。他の入力をすると終了
